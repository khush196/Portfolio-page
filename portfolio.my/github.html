What is GitHub?

GitHub is a platform for hosting and collaborating on code. It uses Git, a version control system, to help developers manage projects, track changes, and work together effectively. GitHub is widely used for open-source contributions and professional development workflows.

Basic Concepts of GitHub

Repositories

A repository (repo) is where your project lives. It contains all your files, code, and a history of changes made to the project.

Branches

Branches are used to develop features, fix bugs, or experiment without affecting the main project. The main branch is the default branch in most projects.

Pull Requests

A pull request allows you to propose changes to a repository. It's a key feature for collaboration and code reviews.
